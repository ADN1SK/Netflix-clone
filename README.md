_**🍿 NETFLUX: A Cinematic UI Experience**_
---

Short version: A sleek, high-performance Netflix clone built entirely with Vanilla HTML, CSS, and JavaScript.

Netflux is an exploration into modern web design, proving that you don't need heavy frameworks to create a beautiful, dynamic, and fully responsive streaming interface. From the custom-built loader to the buttery-smooth horizontal scrolling rows, every pixel is crafted for immersion.

**_✨ Key Features_**
🎬 Pure Poetry in Motion: Custom CSS keyframe animations for the initial loading screen, hero background zoom, and staggered modal entrances.

📱 Flawless Responsiveness: Fluidly adapts from ultra-wide desktop monitors down to mobile screens using modern CSS Flexbox and Grid.

🎨 Dynamic Dark Theme: A deeply rich, sophisticated color palette powered by CSS variables (--bg-dark, --primary-red) for instant global theming.

🖱️ Interactive Micro-Interactions: Hover states that scale movie cards, reveal intricate overlays, and trigger sleek drop-shadows.

🪟 Custom Modal System: A lightweight, vanilla JavaScript modal implementation for viewing movie details without page reloads.

🛠️ Tech Stack
HTML5: Semantic structure and accessibility at the core.

CSS3: Advanced gradients, transitions, custom scrollbars, and variable-driven design.

Vanilla JavaScript: Lightweight DOM manipulation and data rendering.

Environment Agnostic: Whether you are developing on macOS, Windows, or a glorious Fedora Linux machine, this runs natively in any modern browser. 🐧
